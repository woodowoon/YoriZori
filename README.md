# 요리조리
다양한 요리 레시피를 제공해주는 웹 사이트
## 기획의도

&nbsp; 최근 배달비가 부담돼서 요리를 시작하려는 사용자들을 위한, <br>
냉장고 속 재료들을 활용해 무엇을 만들어 먹어야 할지 고민되는 사용자들을 위한,<br>
레시피 제공 웹사이트입니다. <br>

냉장고 속 재료들을 활용하여 긍정적인 경제 효과를 기대할 수 있고, <br>
다양한 사람들에게 요리에 대한 즐거움을 가져다 줄 수 있습니다.
